Level 6: File Sizes & Non-Executable

← Back to Guide

Level Goal

Identify a file in inhere that is non-executable and exactly 1033 bytes in size.

Step‐by‐Step Solution

1. SSH into Level 6

If not already at Level 5, run: 

ssh bandit6@bandit.labs.overthewire.org -p 2220

Enter the Level 5 password (from cat somefile.txt) when prompted.

2. Change Into inhere

List your home directory and navigate: 

ls  
cd inhere

3. Find the File by Size & Type

We want a file that is “non-executable” and exactly 1033 bytes. Use find: 

find . -type f ! -executable -size 1033c

This will output something like ./somefile.

4. Read That File

Use cat to display its contents (the Level 7 password): 

cat somefile

The printed string is the Level 7 password.

Summary & Key Commands

  • ssh bandit6@bandit.labs.overthewire.org -p 2220 : log in as bandit6.
  • cd inhere : enter the inhere directory.
  • find . -type f ! -executable -size 1033c : locate the 1033-byte non-executable file.
  • cat <that-filename> : print the Level 7 password.
