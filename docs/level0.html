Level 0: SSH Login

← Back to Guide

Level 0 Goal

The goal of this level is to log into the Bandit wargame via SSH. Use the following connection details:

  • Host: bandit.labs.overthewire.org
  • Port: 2220
  • Username: bandit0
  • Password: bandit0

Once logged in, open the readme file to find the password for Level 1.

Step-by-Step Solution

1. Check the SSH manual

Use the man command to see how to specify port and username: 

man ssh

In the manual, you'll discover:

  • -p <port> to set a custom port
  • -l <user> to specify the login name

2. Connect via SSH

Run the following command (either form works). This logs you in as bandit0 on port 2220: 


# Option A: use -p and -l flags
ssh -p 2220 -l bandit0 bandit.labs.overthewire.org

# Option B: embed the username directly
ssh bandit0@bandit.labs.overthewire.org -p 2220

When prompted, enter the password: bandit0.

3. View the “readme” file

After logging in, list the files in your home directory: 

ls

You should see a single file named readme. To read its contents: 

cat readme

The output shows the password for Level 1.

Summary & Key Commands

  • man ssh : read the SSH manual to learn flags like -p and -l.
  • ssh bandit0@bandit.labs.overthewire.org -p 2220 : connect via SSH on port 2220.
  • ls : list files in the current directory.
  • cat readme : display the contents of the “readme” file (shows Level 1 password).
