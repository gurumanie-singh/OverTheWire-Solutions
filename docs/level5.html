Level 5: File Types (Human-Readable)

← Back to Guide

Level Goal

Find the only human-readable file in the inhere directory.

Step-by-Step Solution

1. SSH into Level 5

If you're not already at Level 4, run: 

ssh bandit5@bandit.labs.overthewire.org -p 2220

Enter the Level 4 password (from cat .hidden) when prompted.

2. Change Into the inhere Directory

List your home directory and navigate to inhere: 

ls  
cd inhere

3. Identify the Human-Readable File

Use the file command to detect file types. Only one file is “ASCII text”: 

file * | grep "ASCII text"

The output will show something like: 

somefile.txt: ASCII text

4. Read the ASCII-Text File

Once you know the filename (e.g., somefile.txt), display its contents: 

cat somefile.txt

This prints the Level 6 password.

Summary & Key Commands

  • ssh bandit5@bandit.labs.overthewire.org -p 2220 : log in as bandit5.
  • cd inhere : enter the inhere directory.
  • file * | grep "ASCII text" : find which file is human-readable.
  • cat <that-filename> : display its contents to get the Level 6 password.
