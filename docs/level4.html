Level 4: Hidden Files

← Back to Guide

Level Goal

The password for the next level is inside a hidden file in the inhere directory.

Step-by-Step Solution

1. SSH into Level 4

If you aren't already at Level 3, run: 

ssh bandit4@bandit.labs.overthewire.org -p 2220

Enter the Level 3 password when prompted.

2. Change Into the inhere Directory

Once logged in as bandit4, list your home directory: 

ls

You’ll see a folder named inhere. Change into that folder: 

cd inhere

3. List Hidden Files

By default, ls hides files beginning with a dot. Use -a to reveal them: 

ls -a

You should now see a file named .hidden.

4. Read the Hidden File

Use cat to display its contents: 

cat .hidden

This prints the Level 5 password.

Summary & Key Commands

  • ssh bandit4@bandit.labs.overthewire.org -p 2220 : log in as bandit4.
  • cd inhere : enter the inhere directory.
  • ls -a : list all files, including hidden ones (reveals .hidden).
  • cat .hidden : display the Level 5 password.
